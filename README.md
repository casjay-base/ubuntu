# Raspbian Configuration

### Automated Install
```bash
sudo GH=casjay bash -c "$(curl -LsS https://github.com/casjay/ubuntu/raw/master/install.sh)"
```
### Automated update
```bash
sudo update=yes bash -c "$(curl -LsS https://github.com/casjay/ubuntu/raw/master/install.sh)"
```
